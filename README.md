# High Tatras Landing Page

## Description
This is a fully responsive landing page for the High Tatras, created as a capstone project for a Web Development course. The landing page showcases popular activities, attractions, and contact information, using visual storytelling and **responsive design** principles. It is built with **React** and **Tailwind CSS**.

## Features
- Fully responsive layout (desktop, tablet, mobile)
- Navbar with smooth scroll to sections
- Hero section with headline and call-to-action
- Activities section with animated slider
- Attractions section with cards
- Contact section
- Footer

## Technology Stack
- Frontend: React.js
- Styling:Tailwind CSS
- Images / assets: Locally stored in `/public/images`

## Installation
1. Clone the repository:
git clone https://github.com/CatherinneJ/high-tatras-app

2. Navigate to the project folder:
cd high-tatras-app

3. Install dependencies:
npm install

4. Start the development server:
npm start


## Usage

Scroll through the landing page sections (Hero, Activities, Attractions, Contact, Footer)
Hover over buttons to see interactive effects
Observe the animated slider in the Activities section
